# Movie Ticket Booking App
This is a desktop application developed using C# and Windows Forms as part of my academic coursework.

## Overview
The application allows users to log in, browse movie options, and interact with a simple ticket booking interface.

## Features
- User login system
- Password visibility toggle
- Navigation between forms
- Interactive UI for movie selection
- Basic application logic handling

## Technologies
- C#
- Windows Forms (.NET)
- Visual Studio

## What I Learned
- Building desktop applications with Windows Forms
- Handling user input and events
- Designing UI components
- Structuring multi-form applications
